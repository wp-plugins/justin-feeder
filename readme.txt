Via Shortcode Content in einen Artikel oder eine Seite einfügen, der nur wahlweise nur im Feed oder auch nicht im Feed angezeigt wird.


== Description ==

"JustIn Feeder" fügt zwei Shortcodes dem WYSIWYG-Editor ein, die den Inhalt wahlweise nur im Feed oder nur im Artikel (beziehungsweise auf der Seite) anzeigt.

= Einbindung =

Alle Stores in einem Shortcode integriert und können via Button im Editor von WordPress eingefügt werden. Der Aufbau des Shortcodes ist dabei immer der folgende:

[infeed]*content*[/infeed] => nur im Feed anzeigen

[notinfeed]*content*[/notinfeed] => nicht im Feed anzeigen
